### compile

* install Arduino IDE
* create Sketch book
* open「File」->「Sketchbook」->「ArduCAM_OSD」
* focus ArduCAM_OSD.ino tab
* 「Verify」

### charset

config tool has the *.mcm charset file.

can use [max7456 chars made easy](http://www.mylifesucks.de/tools/max7456/) to dump, modify and create charset.
